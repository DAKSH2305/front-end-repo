Sales Dashboard
This project is a web-based Sales Dashboard built with HTML, CSS, and JavaScript that provides an interactive interface for viewing, analyzing, and exporting sales data. The dashboard allows filtering, visualizing trends, inspecting orders, and exporting records for further analysis.

Features
Interactive Filters for region, product, and date range to tailor the sales data displayed.

Key Metrics Cards showing Total Revenue, Total Orders, Average Order Value, and Conversion Rate.

Charts using Chart.js for visualizing sales trends over time and sales distribution by region (line and pie/bar charts).

Top Products List with revenue and quantity-based rankings.

Recent Sales Table with searchable, sortable order records.

CSV/XLSX Export options for filtered sales data.

Responsive Design optimized for desktop and mobile screens
